# Growbox

Проект домашней умной теплицы / гроубокса для томатов.

Цель - постепенно пройти путь от простого мониторинга влажности почвы на ESP32
до полноценного гроубокса с датчиками, историей измерений, уведомлениями,
автополивом, управлением светом и вентиляцией, а также сервером на Ktor.

## Текущий практический фокус

1. Минимальный Ktor backend.
2. `POST /api/v1/telemetry` для приема показаний от ESP32.
3. Таблицы `devices`, `sensors`, `measurements`.
4. Локальный запуск и базовые тесты.
5. MVP-прошивка ESP32 для отправки телеметрии.
