# Фотографии для страницы проекта

Папка для реальных фотографий проекта.

Рекомендуемые имена файлов:

- `hero-growbox.jpg` - большое фото для верхнего блока страницы.
- `esp32.jpg` - ESP32 и подключение датчиков.
- `sensors.jpg` - датчики влажности почвы.
- `prototype.jpg` - макетная сборка или первый горшок.

Сейчас страница использует визуальные заглушки для карточек. Когда появятся
фото, можно заменить блоки `.photo-placeholder` в `docs/index.html` на обычные
`<img src="assets/photos/esp32.jpg" alt="ESP32 с подключенными датчиками">`.
